# drought_t
This project consists in a set of Python routines to perform a temporal analysis of droughts based in the threshold level method (Tallaksen & van Lanen, 2004; van Loon, 2013; Yevjevich, 1967).

## Python dependencies
* gdal
* math
* matplotlib
* numpy
* ogr
* os
* pandas
* pathlib2
* pyEMD
* scipy
* sys
* toml
* xarray
